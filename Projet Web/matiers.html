Matières Étudiées

  • Algorithmique : Structures de données, conception et optimisation d’algorithmes.
  • Programmation : Python, Java, Dart, C++.
  • Base de données : Conception, gestion avec MySQL et MongoDB.
  • Développement Mobile : Flutter, Android Studio.
  • Intelligence Artificielle : Machine learning avec Python (pandas, scikit-learn).
  • Réseaux : Protocoles, sécurité réseau, configuration de serveurs.
  • Ingénierie Logicielle : Méthodes agiles, gestion de projets (Scrum).

Projets Réalisés

Modèle d’apprentissage

Description : Développement d’un modèle de machine learning pour prédire les performances des étudiants à partir de données académiques.

Technologies utilisées : Python, pandas, scikit-learn, matplotlib.

Fonctionnalités :

  • Préparation et nettoyage des données académiques.
  • Utilisation des algorithmes de classification (Random Forest, SVM).
  • Visualisation des résultats avec matplotlib.

Application mobile de gestion d’eau

Description : Développement d'une application mobile pour surveiller et gérer la consommation d'eau domestique.

Technologies utilisées : Flutter, Firebase, Dart.

Fonctionnalités :

  • Affichage en temps réel de la consommation d'eau.
  • Alertes personnalisées pour les seuils de consommation.
  • Statistiques hebdomadaires et mensuelles.
  • Envoi de notifications en cas de fuite détectée.

Application de gestion de parking

Description : Développement d'une solution complète pour la gestion des parkings urbains, incluant une application mobile et une interface de gestion.

Technologies utilisées : Java, SQLite, Android Studio.

Fonctionnalités :

  • Réservation de places de parking en temps réel.
  • Carte interactive pour visualiser les disponibilités.
  • Historique des réservations.
  • Système de paiement intégré via une passerelle sécurisée.
